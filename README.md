# CHAT-LNM

This is a anonymous chatting application

## Getting Started

### Prerequisites

Nodejs, ExpressJS

### Installing

$ git clone https://github.com/chaitanyagiri/CHAT-LNM.git <br>
$ cd CHAT-LNM<br>
$ npm install<br>
$ npm start <br>

## Deployment

After installing you must get something like this
```
listening on *:3000
```
Then go to http://localhost:3000/
